Summarise any Research Paper!

About:

This web app allows you to import any reasearch paper (in PDF file format) and it automatically sumarises it for you. You can select how long you want the summary to be and also, your level of knowledge on the topic. The app will provide a custom summary, by adapting to the requirements set by you.
